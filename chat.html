Dica: você também pode definir via URL: ?sala=geral&nick=Alex
Como funciona?
  1. Este exemplo usa Firebase Firestore para sincronizar as mensagens em tempo real.
  2. Você só precisa criar um projeto no Firebase, habilitar o Firestore e trocar a configuração abaixo.
  3. Para testes rápidos, use regras abertas (somente durante o desenvolvimento). Em produção, restrinja as regras de segurança.
Configurar regras de teste (temporárias) 
// Firestore Security Rules (desenvolvimento — não use em produção)
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true; // ABERTO — apenas para testes
    }
  }
}
